# 📱 BDHelper App  

BDHelper একটি Android অ্যাপ্লিকেশন যা **Heavy Equipment, Transport, Shifting, Rent-a-Car, এবং Harvester Rent** সার্ভিসের জন্য তৈরি।  
এই অ্যাপটি দুই ধরনের ইউজারের জন্য আলাদা আলাদা ড্যাশবোর্ড এবং সুবিধা প্রদান করে।  

---

## 🚀 Features  

### 👤 For Customers  
- সহজে **Transport / Equipment Rent Request** পোস্ট করতে পারবেন  
- সার্ভিস এরিয়াভিত্তিক Vendor/Partner দেখতে পারবেন  
- Notification সিস্টেমের মাধ্যমে নতুন Offer/Status জানতে পারবেন  
- Multi-language support (🇧🇩 বাংলা / 🇬🇧 English)  

### 🤝 For Partners / Vendors  
- Customer-এর পোস্টগুলো realtime এ দেখতে পারবেন  
- Offer পাঠানো, Accept/Reject করা যাবে  
- Own profile এবং services ম্যানেজ করার সুবিধা  
- Referral Code এর মাধ্যমে নতুন ইউজার যুক্ত করার সুযোগ  

---

## 🛠 Tech Stack  

- **Language**: Java, XML  
- **Architecture**: MVVM (Model-View-ViewModel) + DataBinding  
- **Backend**:  
  - Firebase Authentication (Custom SMS Gateway Integration)  
  - Firebase Firestore (Realtime database)  
  - Firebase Cloud Storage (Media storage)  
- **UI**:  
  - Material Design 3 + Custom Components  
  - Dark/Light Theme Support  
  - RecyclerView + CardView + Chips UI  
- **Libraries**:  
  - Glide (Image Loading)  
  - Retrofit / OkHttp (API Integration)  
  - Firebase SDK  
  - Gson / Moshi
